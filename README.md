# Sudoku Solver in C++

Този мини-проект представлява конзолно приложение на C++, което автоматично решава класически пъзели Судоку (9x9). Програмата използва **рекурсивен алгоритъм с връщане назад (Backtracking)**, за да намери и отпечата всички възможни решения на дадена дъска.

---

## ✨ Основни характеристики

- **Пълно изчерпване (Exhaustive Search):** Намира не просто едно, а всички възможни решения на непълно Судоку.
- **Строга валидация:** Гарантира спазването на правилата чрез отделни проверки за всеки ред, колона и вътрешен 3x3 квадрат.
- **Форматиран изход:** Отпечатва намерените решения в удобен за четене ASCII формат (с разделители `|` и `-`).
- **Проверка за уникалност:** В края на изпълнението програмата съобщава дали първоначално зададеното Судоку е валидно (има точно 1 уникално решение) или не.

---

## ⚙️ Как работи алгоритъмът?

Проектът разчита на елегантен рекурсивен подход:

1. Програмата обхожда дъската клетка по клетка (от ляво надясно и от горе надолу).

2. Ако срещне празна клетка (маркирана с `0`), тя се опитва да постави число от `1` до `9`.

3. За всяко поставено число се извиква функцията `isSudokuValid`, която проверява дали правилата на играта са спазени чрез:
   * `areRowsValid` – проверява за повторения в текущия ред.
   * `areColsValid` – проверява за повторения в текущата колона.
   * `areSquaresValid` – проверява за повторения в съответния 3x3 подквадрат.

4. Ако числото е валидно, рекурсията продължава към следващата клетка (`sudokuSolver`).

5. Ако се стигне до задънена улица (няма валидно число за дадена клетка), алгоритъмът се **връща назад (backtracks)**, занулява клетката и пробва следващото възможно число за предишната позиция.

## 🚀 Бърз старт и тестване

За да тестваш програмата със свое собствено Судоку, трябва да редактираш двумерния масив `board` в главната функция main(). Празните полета се отбелязват с 0.

```C++
int board[SIZE][SIZE] =
{
    {3, 0, 6,  5, 0, 8,  4, 0, 0},
    {5, 2, 0,  0, 0, 0,  0, 0, 0},
    // ... останалите редове ...
};
```

---

### Компилиране и стартиране
Ако използваш `g++` в терминал, можеш да компилираш и стартираш кода със следните команди:

```bash
g++ main.cpp -o sudoku_solver
./sudoku_solver
```
